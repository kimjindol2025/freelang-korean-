# 🎯 K-FreeLang 튜토리얼 #1: Hello Korean

**난이도**: 초급

---

## 1️⃣ 가장 간단한 프로그램

한국형 프로그래밍 언어로 첫 번째 프로그램을 작성해봅시다.

```freelang
출력("안녕하세요, K-FreeLang!")
```

**설명**:
- `출력()` - 내장 함수로 값을 출력합니다
- `"..."` - 문자열 리터럴
- 한글로 코드를 작성합니다!

---

## 2️⃣ 변수 사용

```freelang
변수 이름 = "K-FreeLang"
출력("환영합니다, " + 이름 + "!")
```

**주요 개념**:
- `변수` - 변수 선언 (let)
- `=` - 할당 연산자
- `+` - 문자열 연결

---

## 3️⃣ 함수 정의

```freelang
함수 인사(이름: 문자열) -> 공집합 {
  출력("안녕하세요, " + 이름 + "!")
}

인사("K-FreeLang")
인사("한국")
```

**주요 개념**:
- `함수` - 함수 정의 (fn)
- `(이름: 문자열)` - 파라미터 + 타입
- `-> 공집합` - 반환 타입
- `{}` - 함수 본체

---

## 4️⃣ 조건문

```freelang
변수 나이 = 25

만약 나이 >= 20 {
  출력("성인입니다")
} 아니면 {
  출력("미성년자입니다")
}
```

**주요 개념**:
- `만약` - 조건문 (if)
- `>=` - 비교 연산자
- `아니면` - else

---

## 5️⃣ 반복문

```freelang
반복 i = 1; i <= 5; i = i + 1 {
  출력(i)
}
```

출력:
```
1
2
3
4
5
```

**주요 개념**:
- `반복` - for 루프
- `초기화; 조건; 업데이트` - C 스타일 for

## 💡 팁

K-FreeLang은 **완전히 한글**로 프로그래밍할 수 있습니다:
- 변수명, 함수명도 한글 가능
- 영문과 한글 혼합 가능
- 주석도 자유롭게 한글 사용 가능

```freelang
함수 우리말계산(첫번째: 숫자, 두번째: 숫자) -> 숫자 {
  반환 첫번째 + 두번째
}

변수 결과 = 우리말계산(10, 20)
출력(결과)  // 30
```

**즐거운 프로그래밍!** 🎉
