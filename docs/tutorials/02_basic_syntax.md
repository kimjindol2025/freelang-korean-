## 1️⃣ 함수와 변수의 타입

```freelang
변수 이름: 문자열 = "K-FreeLang"
변수 버전: 숫자 = 1.0
변수 활성: 참거짓 = 참

함수 정보표시() -> 공집합 {
  출력("이름: " + 이름)
  출력("버전: " + 버전)
  출력("활성: " + 활성)
}

정보표시()
```

**개념**:
- `변수명: 타입` - 명시적 타입 선언
- `타입 추론` - 초기값으로 자동 추론 가능
- `-> 공집합` - 반환값이 없는 함수

---

## 2️⃣ 상수 vs 변수

```freelang
상수 PI = 3.14159
변수 원의반지름 = 5.0

함수 원의넓이(반지름: 숫자) -> 숫자 {
  반환 PI * 반지름 * 반지름
}

변수 넓이 = 원의넓이(원의반지름)
출력("넓이: " + 넓이)
```

**차이점**:
- `상수` - 초기화 후 변경 불가
- `변수` - 언제든 재할당 가능

## 3️⃣ 조건문 완전 가이드

```freelang
변수 점수 = 85

만약 점수 >= 90 {
  출력("A 학점")
} 아니면 {
  만약 점수 >= 80 {
    출력("B 학점")
  } 아니면 {
    만약 점수 >= 70 {
      출력("C 학점")
    } 아니면 {
      출력("불합격")
    }
  }
}
```

**꿀팁**: else if를 chaining으로 깔끔하게 작성

## 4️⃣ 반복문 패턴들

```freelang
// 기본 for 루프
반복 i = 1; i <= 5; i = i + 1 {
  출력(i)
}

// 배열 순회 (시뮬레이션)
변수 숫자들 = [10, 20, 30]
변수 index = 0
반복 index = 0; index < 3; index = index + 1 {
  출력(숫자들[index])
}

// 합계 계산
변수 합 = 0
변수 n = 1
반복 n = 1; n <= 10; n = n + 1 {
  합 = 합 + n
}
출력("1부터 10까지의 합: " + 합)
```

## 5️⃣ 함수의 고급 사용법

```freelang
// 재귀 함수
함수 팩토리얼(n: 숫자) -> 숫자 {
  만약 n <= 1 {
    반환 1
  }
  반환 n * 팩토리얼(n - 1)
}

출력("5! = " + 팩토리얼(5))

// 여러 파라미터
함수 최대공약수(a: 숫자, b: 숫자) -> 숫자 {
  만약 b == 0 {
    반환 a
  }
  반환 최대공약수(b, a % b)
}

출력("GCD(48, 18) = " + 최대공약수(48, 18))
```

## 🎯 연습 문제

**문제 1**: 1부터 100까지의 짝수만 더하기
```freelang
변수 짝수합 = 0
변수 i = 2
반복 i = 2; i <= 100; i = i + 2 {
  짝수합 = 짝수합 + i
}
출력(짝수합)  // 2550
```

**문제 2**: 구구단 출력
```freelang
변수 단 = 2
반복 단 = 2; 단 <= 9; 단 = 단 + 1 {
  변수 곱 = 1
  반복 곱 = 1; 곱 <= 9; 곱 = 곱 + 1 {
    출력(단 + " x " + 곱 + " = " + (단 * 곱))
  }
}
```

## 💡 Best Practices

✅ **좋은 코드**:
```freelang
함수 사용자인사(이름: 문자열) -> 공집합 {
  출력("안녕하세요, " + 이름 + "!")
}

사용자인사("김jindol")
```

❌ **피해야 할 패턴**:
```freelang
// 변수명이 불명확
함수 a(b: 문자열) -> 공집합 {
  출력(b)
}
```
