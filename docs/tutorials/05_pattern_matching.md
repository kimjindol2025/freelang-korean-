## 1️⃣ 패턴 매칭 (Pattern Matching)

### 기본 개념

패턴 매칭은 값을 분해하고 경우에 따라 다른 코드를 실행하는 강력한 방법입니다.

```
값

↓ (패턴 매칭)

패턴 1 → 동작 1
패턴 2 → 동작 2
기타 → 기본 동작
```

### 기본 문법

```freelang
조건 값 {
  패턴 패턴1 → 결과1
  패턴 패턴2 → 결과2
  패턴 _ → 기본결과
}
```

### 리터럴 패턴

```freelang
사용 string

함수 요일_이름(요일: 숫자) → 문자열 {
  조건 요일 {
    패턴 1 → "월요일"
    패턴 2 → "화요일"
    패턴 3 → "수요일"
    패턴 4 → "목요일"
    패턴 5 → "금요일"
    패턴 6 → "토요일"
    패턴 7 → "일요일"
    패턴 _ → "알 수 없음"
  }
}

변수 오늘 = 요일_이름(3)
출력(오늘)  // "수요일"
```

### 범위 패턴

```freelang
함수 학점_판정(점수: 숫자) → 문자열 {
  조건 점수 {
    패턴 90..100 → "A"
    패턴 80..89 → "B"
    패턴 70..79 → "C"
    패턴 60..69 → "D"
    패턴 _ → "F"
  }
}

출력(학점_판정(85))  // "B"
```

### 와일드카드 & OR 패턴

```freelang
함수 입력_유효성검사(입력: 문자열) → 참거짓 {
  조건 길이(입력) {
    패턴 0 → {
      출력("입력이 비어있습니다")
      반환 거짓
    }
    패턴 1..10 → 반환 참
    패턴 _ → {
      출력("입력이 너무 깁니다")
      반환 거짓
    }
  }
}
```

### 가드 조건

```freelang
함수 숫자_분류(n: 숫자) → 문자열 {
  조건 n {
    패턴 x 만약 x > 0 → "양수"
    패턴 x 만약 x < 0 → "음수"
    패턴 _ → "영"
  }
}

출력(숫자_분류(10))   // "양수"
출력(숫자_분류(-5))   // "음수"
출력(숫자_분류(0))    // "영"
```

### 실전 예제: 상태 머신

```freelang
열거형 서버상태 {
  Starting,
  Running,
  Stopping,
  Stopped,
}

함수 상태_처리(상태: 서버상태) → 문자열 {
  조건 상태 {
    패턴 Starting → "🔄 시작 중..."
    패턴 Running → "✅ 실행 중"
    패턴 Stopping → "⏹️ 종료 중..."
    패턴 Stopped → "⛔ 종료됨"
  }
}

출력(상태_처리(Running))  // "✅ 실행 중"
```

## 2️⃣ 비동기/대기 (Async/Await)

### 기본 개념

비동기 함수는 오래 걸리는 작업(네트워크, 파일 I/O)을 비차단적으로 처리합니다.

```freelang
비동기 함수 fetchData() → 데이터 {
  // 오래 걸리는 작업
  대기 네트워크_요청()
  반환 데이터
}
```

### Async 함수

```freelang
비동기 함수 getData() → 숫자 {
  출력("데이터 받아오는 중...")
  대기 지연(1000)  // 1초 대기
  출력("완료!")
  반환 42
}

// 호출
대기 getData()
```

### 여러 비동기 작업

```freelang
비동기 함수 병렬처리() → 공집합 {
  변수 결과1 = 대기 작업1()
  변수 결과2 = 대기 작업2()
  변수 결과3 = 대기 작업3()

  출력(결과1)
  출력(결과2)
  출력(결과3)
}
```

### 에러 처리

```freelang
비동기 함수 안전한작업() → 문자열 {
  오류처리 {
    변수 데이터 = 대기 fetchData()
    반환 데이터
  } 잡기 오류 {
    출력("오류: " + 오류)
    반환 "기본값"
  }
}
```

### 실전 예제: API 호출

```freelang
사용 json
사용 crypto.hash

비동기 함수 사용자_조회(ID: 숫자) → 맵<문자열, 문자열> {
  // 1. API 호출
  변수 응답 = 대기 HTTP_GET("https://api.example.com/users/" + ID)

  // 2. JSON 파싱
  변수 사용자 = parse(응답)

  // 3. 데이터 검증
  만약 사용자["id"] != ID {
    던지기 "잘못된 응답"
  }

  // 4. 결과 반환
  반환 사용자
}

// 사용
비동기 함수 main() → 공집합 {
  변수 사용자 = 대기 사용자_조회(123)
  출력(사용자["이름"])
}
```

## 3️⃣ 동시실행 (Spawn)

### 기본 개념

여러 작업을 **동시에** 실행합니다 (병렬 처리).

```freelang
동시실행 {
  작업1()   // 동시 실행
  작업2()   // 동시 실행
  작업3()   // 동시 실행
}
```

### 실전 예제: 병렬 다운로드

```freelang
함수 파일_다운로드_병렬() → 공집합 {
  동시실행 {
    함수 download_file1() → 공집합 {
      출력("파일 1 다운로드 시작")
      대기 HTTP_GET("url1")
      출력("파일 1 완료")
    }

    함수 download_file2() → 공집합 {
      출력("파일 2 다운로드 시작")
      대기 HTTP_GET("url2")
      출력("파일 2 완료")
    }

    함수 download_file3() → 공집합 {
      출력("파일 3 다운로드 시작")
      대기 HTTP_GET("url3")
      출력("파일 3 완료")
    }

    download_file1()
    download_file2()
    download_file3()
  }
}
```

### 출력 예:
```
파일 1 다운로드 시작
파일 2 다운로드 시작
파일 3 다운로드 시작
파일 1 완료
파일 2 완료
파일 3 완료
```

## 4️⃣ 동기화 (Sync)

### 기본 개념

한 번에 **하나의 작업만** 실행합니다 (상호 배제).

```freelang
동기화 {
  작업1()   // 먼저 실행
  작업2()   // 작업1 완료 후 실행
  작업3()   // 작업2 완료 후 실행
}
```

### 공유 자원 보호

```freelang
변수 카운터 = 0

함수 카운터_증가() → 공집합 {
  동기화 {
    변수 현재 = 카운터
    카운터 = 현재 + 1  // 안전하게 증가
  }
}

함수 병렬증가() → 공집합 {
  동시실행 {
    카운터_증가()
    카운터_증가()
    카운터_증가()
  }
}
```

## 5️⃣ 종합 예제: 웹 서버 요청 처리

```freelang
사용 json
사용 string

열거형 요청상태 {
  대기중,
  처리중,
  완료,
  오류,
}

구조체 요청 {
  ID: 숫자,
  URL: 문자열,
  상태: 요청상태,
}

비동기 함수 요청_처리(요청: 요청) → 요청 {
  // 상태 머신
  조건 요청.상태 {
    패턴 대기중 → {
      요청.상태 = 처리중

      오류처리 {
        변수 응답 = 대await 네트워크_요청(요청.URL)
        요청.상태 = 완료
      } 잡기 e {
        요청.상태 = 오류
      }
    }

    패턴 처리중 → 출력("이미 처리 중...")
    패턴 완료 → 출력("이미 완료됨")
    패턴 오류 → 출력("오류 발생")
  }

  반환 요청
}

비동기 함수 대량요청처리(요청들: 배열<요청>) → 공집합 {
  동시실행 {
    반복 변수 i = 0; i < 길이(요청들); i = i + 1 {
      변수 처리됨 = 대기 요청_처리(요청들[i])
      출력("요청 " + 처리됨.ID + " 처리됨")
    }
  }
}
```
