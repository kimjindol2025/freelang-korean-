# K-FreeLang Standard Library (K-StdLib) API Reference

**버전**: v1.0 (Week 2-3)
**마지막 업데이트**: 2026-03-27

---

## 📚 모듈 가이드

K-FreeLang 표준 라이브러리는 7개 핵심 모듈과 3개 암호화 모듈로 구성됩니다.

```
사용 math      // 수학 연산
사용 array     // 배열 조작
사용 string    // 문자열 처리
사용 object    // 객체 조작
사용 json      // JSON 직렬화
사용 io        // 파일 I/O
사용 date      // 날짜/시간

사용 crypto.hash       // SHA-256, HMAC, 암호 해싱
사용 crypto.aes        // AES-256, ARIA-256, SEED-128
사용 crypto.otp        // TOTP, HOTP, 2FA
사용 crypto.encoding   // Base64, Hex, Base32
사용 crypto.tls        // 인증서, 세션, 공개키 암호화
```

## 🔢 math 모듈

기본적인 수학 연산을 제공합니다.

### 함수 목록

| 함수 | 인자 | 반환값 | 설명 |
|------|------|--------|------|
| `절대값(n)` | `숫자` | `숫자` | 절댓값 |
| `최대값(a, b)` | `숫자, 숫자` | `숫자` | 최댓값 |
| `최소값(a, b)` | `숫자, 숫자` | `숫자` | 최솟값 |
| `거듭제곱(a, b)` | `숫자, 숫자` | `숫자` | a^b |
| `제곱근(n)` | `숫자` | `숫자` | √n |
| `올림(n)` | `숫자` | `숫자` | 천장 함수 |
| `내림(n)` | `숫자` | `숫자` | 바닥 함수 |
| `반올림(n)` | `숫자` | `숫자` | 가장 가까운 정수 |

### 예제

```freelang
사용 math

변수 값 = -15
출력(절대값(값))        // 15
출력(최대값(10, 20))    // 20
출력(거듭제곱(2, 10))   // 1024
```

### 함수 목록

| 함수 | 인자 | 반환값 | 설명 |

| `합계(arr)` | `배열<숫자>` | `숫자` | 합 |
