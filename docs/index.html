💡 코드 예제

1️⃣ Hello World

함수 main() → 공집합 {
  출력("안녕하세요, K-FreeLang!")
}

main()

2️⃣ 함수 & 변수

함수 add(a: 숫자, b: 숫자) → 숫자 {
  반환 a + b
}

변수 결과 = add(10, 20)
출력(결과)  // 30

3️⃣ 비동기 프로그래밍 ⭐

비동기 함수 데이터_불러오기() → 문자열 {
  대기 지연(1000)
  반환 "데이터"
}

비동기 함수 main() → 공집합 {
  변수 데이터 = 대기 데이터_불러오기()
  출력(데이터)
}

4️⃣ 개인정보 보호 (PIPA)

변수 주민번호 = "123456-1234567"

// 유효성 검증
변수 유효 = RRN_유효성검증(주민번호)

// 마스킹
변수 마스크됨 = RRN_마스킹(주민번호)
// 결과: "123456-****67"

🚀 시작하기

K-FreeLang을 지금 바로 시작하세요!

1. 저장소 클론

git clone https://github.com/kimjindol2025/freelang-korean-.git
cd freelang-korean-independent
npm install

2. 첫 프로그램 작성

cat > hello.free << 'EOF'
함수 main() → 공집합 {
  출력("안녕하세요!")
}

main()
EOF

3. 실행

npm run run hello.free
