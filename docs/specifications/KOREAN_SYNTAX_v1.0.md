## ✨ 설계 원칙

### 1. 국립국어원 표준어 준수
- ✅ 정식한글 사용
- ❌ 영어 차용 최소화
- 예: "비동기" (O) vs "에이싱크" (X)

### 2. 동사형 + 목적어 구조
```
함수 [목적어]_[동사형](파라미터) -> 반환타입
```
- `함수` (함수 정의)
- `변수` (변수 선언)
- `반환` (값 반환)

### 3. 언더스코어(_) 기반 연결
- 다중 단어: `동시_실행`, `뮤텍스_잠금`
- 한글 가독성 우선

### 4. 한국 문화 반영
- 한국 표준 암호화 (ARIA, SEED)
- 한국 고유 신원 체계 (주민번호, 사업자번호)
- 한국식 법령 준수 (PIPA, ISMS, PCI-DSS)
