# 📚 K-FreeLang 한글 문법 v1.0

**한국형 독립 프로그래밍 언어 - 한글 문법 명세**

---

## 🎯 개요

K-FreeLang은 **한국 개발자를 위해 설계된 프로그래밍 언어**입니다.

기본 문법을 **완전히 한글로** 작성할 수 있으며, 동시에 영문 키워드도 지원하여 국제적 호환성을 유지합니다.

### 핵심 철학
> **"한국이 만든 언어가, 한국 코드로, 한국을 검증한다"**

- 🇰🇷 한글 우선 (영문 병행)
- 🔐 보안 규제 내재화 (PIPA, ISMS, ARIA/SEED)
- 📖 한국 개발자 중심 설계
- 🚀 완전 독립된 런타임

| 순위 | 한글 | 영문 | 토큰 타입 | 빈도 | 용도 |

| 3 | 만약 | `if` | IF | 567 | 조건문 |
| 4 | 함수 | `fn`/`def` | FN | 178 | 함수 정의 |
| 5 | 반복 | `for`/`while` | FOR/WHILE | 234 | 반복문 |
| 6 | 아니면 | `else` | ELSE | 156 | 조건 분기 |
| 7 | 타입 | `type` | TYPE | 89 | 타입 정의 |
| 8 | 상수 | `const` | CONST | 45 | 상수 선언 |

### 조건문

```freelang
// 한글 문법
변수 점수 = 85

만약 점수 >= 90 {
  출력("A 학점")
} 아니면 만약 점수 >= 80 {
  출력("B 학점")
} 아니면 {
  출력("C 학점 이하")
}

// 영문 문법 (병행)
if score >= 90 {
  print("Grade A")
} else if score >= 80 {
  print("Grade B")
} else {
  print("Grade C or below")
}
```

### 반복문

```freelang
// for 반복 (한글)
반복 i = 1; i <= 5; i = i + 1 {
  출력(i)
}

// while 반복 (한글)
변수 x = 0
반복 x < 10 {
  출력(x)
  x = x + 1
}

// 영문 문법 (병행)
for i = 1; i <= 5; i++ {
  print(i)
}

while x < 10 {
  print(x)
  x++
}
```

## 📦 타입 시스템

### 기본 타입

| 한글 | 영문 | 설명 | 예제 |
|------|------|------|------|
| 숫자 | `number` | 정수/실수 | `변수 x = 42` |
| 문자열 | `string` | 텍스트 | `변수 이름 = "김프리"` |
| 참거짓 | `bool` | 불린 | `변수 참 = 참` |
| 정수 | `i32`/`i64` | 부호있는 정수 | `변수 a: i32 = -5` |
| 부호없는정수 | `u32`/`u64` | 부호없는 정수 | `변수 b: u32 = 10` |
| 실수 | `f32`/`f64` | 부동소수점 | `변수 pi: f64 = 3.14` |
| 문자 | `char` | 단일 문자 | `변수 ch = 'A'` |
| 공집합 | `void` | 값 없음 | `함수 fn() -> void {}` |

## 🔗 혼합 문법 (Mixed Mode)

K-FreeLang은 **한글과 영문 키워드를 자유롭게 섞어 사용**할 수 있습니다.

```freelang
// 완전 한글
함수 계산(x: 숫자) -> 숫자 {
  변수 결과 = x * 2
  반환 결과
}

// 완전 영문
fn calculate(x: number) -> number {
  var result = x * 2
  return result
}

// 혼합 (권장하지 않음, 하지만 지원)
함수 process(data: array) -> 숫자 {
  let sum = 0
  for i in data {
    sum += i
  }
  return sum
}
```

## 🎨 문법 규칙

### 식별자 (변수명, 함수명)

✅ **허용**:
- 한글: `변수 이름 = "값"`
- 영문: `let name = "value"`
- 혼합 (권장 안함): `let 이름 = "값"`
- 숫자: `변수 점수2 = 95`
- 언더스코어: `변수 최종_점수 = 100`

❌ **불허**:
- 공백: `변수 최종 점수 = 100` ❌
- 특수문자: `변수 점수$ = 95` ❌
- 숫자로 시작: `변수 2점수 = 95` ❌

---

### 문자열

```freelang
// 큰따옴표 (권장)

변수 문자 = '한'

변수 줄바꿈 = "첫번째\n두번째"
변수 탭 = "들여쓰기\t텍스트"
변수 따옴표 = "큰\"따옴표"

// 문자열 결합
변수 인사 = "안녕" + ", " + "프리랭"
```

## 🔐 한국 규제 표준 통합

K-FreeLang은 컴파일 타임에 한국 규제를 검증합니다.

### 예제: PIPA (개인정보보호법) 준수

```freelang
@comply(PIPA: true)
함수 고객정보저장(이름: 문자열, 계좌: 문자열) -> 참거짓 {
  // 암호화 필수 (PIPA)
  암호화된이름 = ARIA_암호화(이름, 마스터키)
  암호화된계좌 = SEED_암호화(계좌, 마스터키)

  // 저장소에 저장
  저장(암호화된이름, 암호화된계좌)

  반환 참
}
```

**컴파일 타임 검증**:
```
✅ 개인정보 암호화됨
✅ ARIA/SEED 사용 (국내 표준)
✅ 감시 로그 포함
✅ PIPA 준수 OK
```
