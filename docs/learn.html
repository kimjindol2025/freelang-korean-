1 개념 이해하기: K-FreeLang이란?

초급
🎯 목표: K-FreeLang의 기본 개념과 특징을 이해합니다.

배울 내용

  • K-FreeLang은 무엇인가?
  • 한글 프로그래밍의 장점
  • 47개 한글 키워드 소개
  • 자가호스팅(Self-Hosting) 개념

핵심 개념:

  • K-FreeLang = 한국 개발자를 위한 완전 독립형 프로그래밍 언어
  • 한글 기반 문법 → 직관적인 코드 작성
  • PIPA/ISMS 준수 → 보안과 규제 준수 내재화
  • 자가호스팅 → K-FreeLang으로 K-FreeLang 자체를 컴파일

2 개발 환경 준비하기

초급
🎯 목표: K-FreeLang 개발 환경을 설정합니다.

배울 내용

  • 저장소 클론하기
  • 의존성 설치 (npm install)
  • 컴파일러 빌드
  • 첫 실행 확인

설정 단계:

# 1. 저장소 클론 git clone https://github.com/kimjindol2025/freelang-korean-.git cd freelang-korean-independent # 2. 의존성 설치 npm install # 3. 컴파일러 빌드 npm run build # 4. 첫 실행 npm run hello

확인 사항:

  • ✅ Node.js 18.x 이상
  • ✅ npm 9.x 이상
  • ✅ Git 설치

4 변수와 데이터 타입

초급
🎯 목표: 변수 선언과 다양한 데이터 타입을 학습합니다.

배울 내용

  • 변수 선언 (변수 키워드)
  • 기본 타입 (숫자, 문자열, 참거짓)
  • 배열과 맵
  • 타입 주석

변수 선언:

변수 이름 = "김철수" 변수 나이: 숫자 = 25 변수 활성: 참거짓 = 참 변수 점수들: 배열<숫자> = [90, 85, 95] 변수 정보: 맵<문자열, 공집합> = {}

기본 타입:

  • 숫자 → 정수, 소수
  • 문자열 → 텍스트
  • 참거짓 → true/false
  • 배열<T> → 리스트
  • 맵<K,V> → 딕셔너리

5 함수 정의와 호출

초급
🎯 목표: 함수를 정의하고 사용하는 방법을 배웁니다.

배울 내용

  • 함수 정의 문법
  • 매개변수와 반환값
  • 함수 호출

함수 정의:

함수 더하기(a: 숫자, b: 숫자) → 숫자 { 반환 a + b } 함수 인사(이름: 문자열) → 문자열 { 반환 "안녕하세요, " + 이름 } // 함수 호출 변수 결과 = 더하기(10, 20) 변수 메시지 = 인사("철수")

함수 문법 규칙:

  • 함수 이름(매개변수: 타입) → 반환타입 { ... }
  • 매개변수 타입 필수
  • 반환 타입 필수
  • 공집합 = void (반환값 없음)

6 제어 흐름: if/else와 반복문

초급
🎯 목표: 조건문과 반복문을 사용합니다.

배울 내용

  • 조건문 (if/else)
  • 반복문 (for)
  • 반복 제어 (break, continue)
  • 중첩 제어

조건문:

만약 나이 >= 18 { 출력("성인입니다") } 아니면 { 출력("미성년자입니다") }

반복문:

// for 루프 반복 변수 i = 0; i < 10; i = i + 1 { 출력(i) } // break와 continue 반복 변수 i = 0; i < 10; i = i + 1 { 만약 i == 5 { 중단 // break } }

비동기 함수:

비동기 함수 데이터_불러오기() → 문자열 { // 1초 대기 대기 지연(1000) 반환 "데이터 로드 완료" } 비동기 함수 main() → 공집합 { 변수 결과 = 대기 데이터_불러오기() 출력(결과) }

에러 처리:

오류처리 { 변수 데이터 = 대기 데이터_불러오기() } 잡기 오류 { 출력("오류 발생: " + 오류) }

9 동시성 프로그래밍: Mutex와 Channel

고급
🎯 목표: Mutex와 Channel로 동시 처리를 구현합니다.

배울 내용

  • Mutex (상호배제)
  • Channel (채널)
  • 동시실행 (goroutine 유사)
  • 데이터 경쟁(Race Condition) 방지

Mutex 사용:

뮤텍스 카운터 { 숫자 = 0 } 함수 증가() → 공집합 { 잠금(카운터) { 카운터.값 = 카운터.값 + 1 } }

Channel 사용:

채널 메시지<문자열>(1) 함수 발신자() → 공집합 { 메시지 <- "안녕!" } 함수 수신자() → 공집합 { 변수 msg = <- 메시지 출력(msg) } 동시실행 { 발신자() 수신자() }

배울 내용

예제: 주민등록번호 검증 시스템

// PIPA 준수: 주민등록번호 검증 함수 주민번호_검증(rrn: 문자열) → 참거짓 { // RRN_유효성검증() 사용 반환 참 } 함수 주민번호_마스킹(rrn: 문자열) → 문자열 { // RRN_마스킹() 사용 반환 "123456-****67" } // ISMS 준수: 감사 로그 함수 감시(사용자: 문자열, 행동: 문자열) → 공집합 { 출력("[" + 사용자 + "] " + 행동) }
